# 付录 (PayCord)

**付录** 是一款秉持“无感、隐私至上”原则开发的开源记账程序。

在碎片化消费的时代，记账不应成为生活的负担，而应是理清财务脉络的静默助手。我们致力于在保护用户隐私的前提下，通过技术手段降低记账门槛，帮助每一位用户重新获得对消费压力的感知能力。

---

### 核心理念

* **多维记账架构**：支持 **无账户模式** 与 **多账户体系**。无论是不愿受碎片化账户束缚的简易派，还是需要精细化管理多渠道资产的投资者，都能在此找到舒适的记账节奏。
